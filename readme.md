# rust_dev

For learning Rust.

## VSCode extension

 - Dev Containers(Visual Studio Code Remote - Containers)

## Usage

### Run

```sh
cd project
```

and

```sh
cargo run
```

### Lint

```sh
cargo clippy
```

### Format

```sh
cargo fmt
```

If the `rustfmt` does not exist...

```sh
rustup component add rustfmt
```
